# 💧 水飲みリマインド

PC作業中に水分補給を促してくれる、かわいいマリンテイストのデスクトップアプリです。  
設定可能な間隔（5～45分）で通知ポップアップが表示され、飲水回数をコップのビジュアルで確認できます。

---

## 画面イメージ

| メイン画面 | 通知ポップアップ |
|---|---|
| 飲水回数・コップビジュアル・残り時間を表示 | 画面右下に表示（25秒後に自動消去） |

---

## 機能

- ⏱ **カスタマイズ可能なタイマー** — 5～45分の範囲で通知間隔を設定可能（デフォルト20分）。起動後すぐにカウントダウン開始し、次の通知までの残り時間をリアルタイム表示
- 🔔 **通知ポップアップ** — 設定時間経過すると画面右下にポップアップを表示（25秒後に自動消去）
- ✔ **「飲んだ」ボタン** — ポップアップまたはメイン画面の「飲んだ」を押すと飲水回数が増え、タイマーがリセット
- ⏸ **「あとで」ボタン** — 5分後に再度通知を表示（通常時）
- 🥤 **コップビジュアル** — テイクアウトカップ風のデザインで、飲水回数に応じて水位がなめらかにアニメーションしながら上昇（8回で満水）。波のゆらぎやリップルエフェクトで動きのある表現
- 💬 **ランダムメッセージ** — 通知ポップアップに毎回異なる応援メッセージを表示。無視回数に応じてメッセージが優しく変化
- 🎨 **複数のカラーテーマ** — 7種類のテーマから選択可能（水色、ピンク、黄色、緑、紫、白、黒）
- 🪶 **軽量・常駐動作** — 外部サービス・データベース不使用。ウィンドウを最小化してもバックグラウンドでタイマーが動作し続けます

## 使い方

### 基本的な使い方

1. アプリを起動するとメイン画面が表示され、タイマーが自動でスタートします
2. **設定時間経過**すると、画面右下に通知ポップアップが表示されます
3. 水を飲んだら **「✔ 飲んだ」** ボタンを押してください
   - 飲水回数が1増えます
   - タイマーがリセットされ、次のカウントが始まります
   - コップの水位がアニメーションしながら上がります
4. 水を飲まない場合は **「あとで」** でポップアップを閉じられます
   - 5分後に再度通知が表示されます
5. ポップアップは **一定時間経過すると半透明になります**
6. メイン画面の **「💧 飲んだ」** ボタンからも、通知を待たずにいつでも記録できます

### 画面操作

- **メイン画面のドラッグ移動** — メイン画面をドラッグして好きな位置に移動できます
- **右上の「✕」ボタン** — アプリを終了できます
- **歯車アイコン** — 設定ダイアログを開きます

### 設定

歯車アイコンをクリックして設定ダイアログを開くと、以下の項目を変更できます：

- **通知間隔** — 5～45分の範囲で5分刻みで設定可能（デフォルト20分）
- **カラーテーマ** — 7種類のテーマから選択可能

設定は自動的に保存され、次回起動時に反映されます。

### 無視時の挙動

通知を無視して複数回閉じられた場合：
- メッセージ内容がより優しく強調されます
- ポップアップの枠線が強調表示されます
- 最終警告時には「⚠ 閉じると水位が少し下がります」という警告が表示されます
- 「閉じる」ボタンを押すと、水位が少し下がるペナルティが適用されます

---

## 注意事項

- アプリを終了すると飲水回数はリセットされます（データは保存されません）
- 再起動時は回数・タイマーともに初期状態から始まります
- 設定（通知間隔・カラーテーマ）のみ保存されます
