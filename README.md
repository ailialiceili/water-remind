# 💧 水飲みリマインド

PC作業中に水分補給を促してくれる、かわいいマリンテイストのデスクトップアプリです。  
30分ごとに通知ポップアップが表示され、飲水回数をコップのビジュアルで確認できます。

---

## 画面イメージ

| メイン画面 | 通知ポップアップ |
|---|---|
| 飲水回数・コップビジュアル・残り時間を表示 | 画面右下に30分ごとに表示 |

---

## 機能

- ⏱ **30分タイマー** — 起動後すぐにカウントダウン開始。次の通知までの残り時間をリアルタイム表示
- 🔔 **通知ポップアップ** — 30分経過すると画面右下にポップアップを表示（25秒後に自動消去）
- ✔ **「飲んだ」ボタン** — ポップアップまたはメイン画面の「飲んだ」を押すと飲水回数が増え、タイマーがリセット
- 🥤 **コップビジュアル** — テイクアウトカップ風のデザインで、飲水回数に応じて水位がなめらかにアニメーションしながら上昇（8回で満水）
- 💬 **ランダムメッセージ** — 通知ポップアップに毎回異なる応援メッセージを表示
- 🪶 **軽量・常駐動作** — 外部サービス・データベース不使用。ウィンドウを最小化してもバックグラウンドでタイマーが動作し続けます

---

## 動作環境

- **OS**: Windows
- **Python**: 3.10 以上
- **必要パッケージ**: PyQt6

## セットアップ

### 1. 初回セットアップ

`run_setup.bat` をダブルクリックして実行してください。  
仮想環境の作成と必要なパッケージ（PyQt6）のインストールが自動で行われます。

```
run_setup.bat をダブルクリック
```

> ※ Python がインストールされていない場合は、先に [python.org](https://www.python.org/downloads/) からインストールしてください。

## 起動方法

`run_app.bat` をダブルクリックして実行してください。

```
run_app.bat をダブルクリック
```

---

## 使い方

1. アプリを起動するとメイン画面が表示され、タイマーが自動でスタートします
2. **30分経過**すると、画面右下に通知ポップアップが表示されます
3. 水を飲んだら **「✔ 飲んだ」** ボタンを押してください
   - 飲水回数が1増えます
   - タイマーがリセットされ、次の30分のカウントが始まります
   - コップの水位がアニメーションしながら上がります
4. 水を飲まない場合は **「✕ 閉じる」** でポップアップを閉じられます（タイマーはリセットされません）
5. ポップアップは **25秒後に自動で消えます**
6. メイン画面の **「💧 飲んだ」** ボタンからも、通知を待たずにいつでも記録できます
7. メイン画面はドラッグして好きな位置に移動できます
8. 右上の **「✕」** ボタンでアプリを終了できます

---

## 注意事項

- アプリを終了すると飲水回数はリセットされます（データは保存されません）
- 再起動時は回数・タイマーともに初期状態から始まります
